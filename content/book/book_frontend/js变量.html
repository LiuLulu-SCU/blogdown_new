19 JS变量

19.1 变量的定义、声明和赋值

定义

通俗：变量是用于存放数据的容器。 晦涩：变量是程序在内存中申请的一块用来存放数据的空间。

变量在使用时分为两步： 1. 声明变量 2. 赋值

19.1.1 声明变量

<body>
    <script>
        // 声明了一个age变量
        var age;
    </script>
</body>
  • var 是一个 JS关键字，用来声明变量( variable 变量的意思 )。使用该关键字声明变量后，计算机会自动为变量分配内存空间，
  • age 是程序员定义的变量名，我们要通过变量名来访问内存中分配的空间

19.1.2 变量赋值

<body>
    <script>
        // 赋值，把值存入变量中
        age = 10
    </script>
</body>

19.1.3 变量初始化

我们可以一步完成变量的声明和赋值，被称为变量的初始化。

<body>
    <script>
        // 变量的初始化
        var my_name = 'pink老师';
        console.log(my_name);
    </script>
</body>

19.2 变量语法扩展

包括：更新变量、同时声明多个变量、声明变量特殊情况

19.2.1 更新变量

一个变量被重新复赋值后，它原有的值就会被覆盖，变量值将以最后一次赋的值为准。

<body>
    <script>
        var age = 18;
        age = 81 //最后的age为81
    </script>
</body>

19.2.2 同时声明多个变量

联系CSS可以同时创建多个属性。JS也同时声明多个变量。

同时声明多个变量时，只需要写一个 var， 多个变量名之间使用英文逗号隔开。

<body>
    <script>
        var age = 10, name = 'LLL', sex =‘男
    </script>
</body>

19.2.3 声明变量特殊情况

表4.1: JS声明变量特殊情况
情况 说明 结果
var age ; console.log (age); 只声明 不赋值 undefined
console.log(age) 不声明 不赋值 直接使用 报错
age = 10; console.log (age); 不声明 只赋值 10

详见：示例

19.3 变量的命名规范

  • 由字母(A-Za-z)、数字(0-9)、下划线(_)、美元符号( $ )组成，如：usrAge, num01, _name
  • 严格区分大小写。var app; 和 var App; 是两个变量
  • 不能 以数字开头。 18age 是错误的
  • 不能是关键字、保留字。挖个坑，后面会补上是啥意思。
  • 遵守驼峰命名法。首字母小写，后面单词的首字母需要大写。 myName

在R和Python中，变量一般使用下划线命名法：如：my_name。
